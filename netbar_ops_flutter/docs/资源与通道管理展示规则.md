# 资源管理与通道管理展示规则

> 更新时间：2026-01-31

---

## 一、角色定义

| 角色 | 判断条件 | 说明 |
|------|----------|------|
| **SuperAdmin** | `group_id == 0 或 null` | 总部管理员 |
| **Admin** | `group_id > 0 && is_manager == true` | 分部管理员 |
| **普通用户** | `group_id > 0 && is_manager == false` | 普通用户 |

---

## 二、【资源管理】模块

### 2.1 左侧菜单

固定显示三个区域：
1. **总公司资源**
2. **网吧资源**
3. **共享资源**

### 2.2 文件数据展示

| 区域 | API 接口 | 数据筛选条件 |
|------|----------|-------------|
| **总公司资源** | `GET /api/file/view` | `data.files` 中 `group_id == 0` 的数据 |
| **网吧资源** | `GET /api/file/view` | `data.files` 中 `group_id != 0` 的数据 |
| **共享资源** | `GET /api/file/shared` | 接口返回的所有数据 |

### 2.3 启动项数据展示

| 区域 | 是否显示启动项 Tab | API 接口 | 说明 |
|------|-------------------|----------|------|
| **总公司资源** | ❌ 不显示 | - | 暂时不显示启动项 |
| **网吧资源** | ✅ 显示 | `GET /api/tactic` | 显示当前网吧的启动项（根据网吧ID过滤） |
| **共享资源** | ❌ 不显示 | - | 暂时不显示启动项 |

**启动项 API 参数示例：**
```
GET /api/tactic?page=1&per_page=100&keyword=xxx&type=merchant
```

**前端过滤逻辑：**
- 从 `/api/tactic` 响应中获取所有启动项
- 根据当前选中的网吧ID过滤：`item.merchantId == currentNetbarId`
- 仅显示属于当前网吧的启动项

### 2.4 文件编辑权限

| 区域 | SuperAdmin | Admin | 普通用户 |
|------|------------|-------|----------|
| **总公司资源** | ✅ 可编辑 | ❌ 只读 | ❌ 只读 |
| **网吧资源** | ✅ 可编辑 | ✅ 可编辑 | ❌ 只读 |
| **共享资源** | ✅ 可编辑 | ✅ 可编辑 | ⚠️ 仅可编辑自己上传的文件（`user_id == 当前用户 id`） |

### 2.5 启动项编辑权限

| 区域 | SuperAdmin | Admin | 普通用户 | 启用/禁用 |
|------|------------|-------|----------|-----------|
| **总公司资源** | ❌ 只读 | ❌ 只读 | ❌ 只读 | ❌ 不可操作 |
| **网吧资源** | ❌ 只读 | ❌ 只读 | ❌ 只读 | ❌ 不可操作 |
| **共享资源** | - | - | - | 无启动项 |

> **注意**：资源管理中所有模块、所有人都不可编辑/启用/禁用启动项，仅可查看。

## 六、API 接口说明

### 6.1 文件列表

```http
GET /api/file/view
```

响应结构：
```json
{
  "code": 0,
  "data": {
    "files": [
      {
        "id": 1,
        "name": "文件名",
        "group_id": 0,  // 0=总公司, >0=网吧
        "user_id": 123,
        ...
      }
    ],
    "merchants": [...]
  }
}
```

### 6.2 共享文件列表

```http
GET /api/file/shared
```

### 6.3 启动项列表

```http
GET /api/tactic?page=1&per_page=100&keyword=xxx&type=merchant
```

响应结构：
```json
{
  "code": 0,
  "data": {
    "paginator": {
      "data": [
        {
          "id": 网吧ID,
          "name": "网吧名称",
          "tactics": [
            {
              "id": 策略ID,
              "startup": {...},
              "locales": [...],
              ...
            }
          ]
        }
      ]
    }
  }
}
```
